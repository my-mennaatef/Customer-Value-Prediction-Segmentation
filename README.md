# Customer-Value-Prediction-Segmentation
Project analyzes 200K+ e-commerce interactions to build customer segments and predict value. It includes session creation, RFM features, behavior analysis, funnel metrics, category insights, and ML models for purchase probability and lifetime value to support data-driven marketing.
